# GoDoctor-Healthcare-Management-System
GoDoctor is a healthcare web application that connects patients, doctors, and support staff on one platform. It allows appointment booking, patient record management, and secure login with role-based access. Built using React, Node.js, and MongoDB, it also supports real-time communication.
